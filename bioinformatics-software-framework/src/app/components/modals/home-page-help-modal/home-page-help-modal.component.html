BioWorkflow Help Desk

Taking user input to the workflow

  1. Click on User Input
  2. Add it to the Canvas
  3. Click on Sequence
    • Add a test sequence
    • Add multiple sequences for MSA
  4. Paste your sequence

Connecting components in the work flow

  1. Add two or more components
  2. Click on one component
  3. Click on the other component

Visualizing resulting DNA sequences

  1. Add User Input component
  2. Add Clustal Omega components (For MSA)
  3. Connect to Visualize component
  4. Enter the sequences in the User Input
  5. Press Execute
  6. Visualization can be seen at the bottom
